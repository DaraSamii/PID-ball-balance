# PID-ball-balance

this repository contain 3D cads, codes and arduino schematics for makinh 3d PID balance project.

![ScreenShot](photo.jpg)

# list of objects used for the project
* Ardoino Uno
* GP2Y0E3 sensor
* Servo motor
* balsa wood
